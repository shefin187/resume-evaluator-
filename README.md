# AI Resume Screening System

An AI-powered Resume Screening System that parses PDF and DOCX resumes, extracts structured candidate information using Groq LLM and Pydantic, analyzes job descriptions, compares resumes against job requirements, and ranks candidates based on their overall match score.

## Features
- Parse PDF and DOCX resumes
- Extract structured resume information
- Analyze job descriptions
- AI-powered resume matching
- Candidate ranking by score
- JSON-based structured outputs

## Tech Stack
- Python
- Groq API
- Pydantic
- python-dotenv
- PyPDF
- python-docx
